# Tweet Text Classification
This project delves into the emotional pulse of Twitter by classifying tweets based on sentiment.

## Libraries Used
- Nltk
- Spacy
- Matplotlib
- Pandas
- WordCloud
- Seaborn
- Sklearn

## Different Steps
1. Data Exploration 
2. Data Preprocessing
3. Data Splitting
4. Model Building (Logistic Regression Model)
